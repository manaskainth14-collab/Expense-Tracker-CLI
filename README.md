# 💰 Expense Tracker

A simple command-line **Expense Tracker** built using Python to practice core programming concepts such as lists, dictionaries, loops, functions, conditional statements, and user input.

---

## 📌 Features

- ➕ Add new expenses
- 📋 View all recorded expenses
- 💵 Calculate total expenses
- 🚪 Menu-driven command-line interface

---

## 🛠️ Technologies Used

- Python 3

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/manaskainth14-collab/Expense-Tracker-CLI.git
```

### 2. Navigate to the project folder

```bash
cd Expense-Tracker-CLI
```

### 3. Run the program

```bash
python Expense_tracker.py
```
